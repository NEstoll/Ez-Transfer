Usage Instructions: The home screen of the app shows the file selection. If proper permissions are not yet enabled, you need to allow them from settings for the app before opening it. From the home screen, press the select files button. This brings you to the file selection for the phone. Once selected, you will need to press next to be taken to the device selection screen. Once a device is selected, the transfer progress screen shows when the transfer is complete.

Known Bugs: No prompt for permissions
